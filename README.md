# 🧠 Juego de Memoria Interactivo

Juego clásico de memoria desarrollado con HTML5, CSS3 (Tailwind CSS) y JavaScript puro. Pon a prueba tu capacidad de retención encontrando todos los pares de cartas iguales.

![Demo del juego](https://via.placeholder.com/600x400?text=Captura+del+Juego)

## 🚀 Características

- Tablero de 4x4 con 8 pares de iconos (🐱, 🐶, 🐟, 🐦, 🍎, 👑, ⭐, ❤️).
- Animaciones 3D estilo "flip card" al voltear cada carta.
- Contador de movimientos y pares encontrados.
- Mensaje de victoria con número total de movimientos.
- Botón para reiniciar la partida en cualquier momento.
- Totalmente responsive, adaptable a móviles y escritorio.

## 🛠️ Tecnologías utilizadas

- **HTML5** – Estructura semántica.
- **Tailwind CSS** – Estilos rápidos y responsive.
- **JavaScript (ES6)** – Lógica del juego, manejo de estados y eventos.
- **Font Awesome** – Iconografía atractiva.

## 📦 Instalación y uso

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador web favorito.
3. ¡Empieza a jugar!

No requiere servidor ni dependencias externas (solo conexión a internet para cargar Tailwind y Font Awesome desde CDN).

## 🎮 Cómo jugar

- Haz clic en cualquier carta para voltearla y ver su icono.
- Luego haz clic en otra carta para buscar su par.
- Si coinciden, las cartas permanecen descubiertas y suman un punto.
- Si no coinciden, se volverán a ocultar después de 0.8 segundos.
- El objetivo es emparejar todas las cartas con el menor número de movimientos posible.
